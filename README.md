# Retro Snake Game (C++ / Web)

This project includes a prebuilt web version in the `web/` folder.

## Public URL (Stable)

After pushing to `main`, GitHub Pages will publish the web build at:

`https://ericpham7.github.io/Retro-Snake-Game-C-/`

Deployment is handled by:

`.github/workflows/deploy-pages.yml`

## Public URL (Temporary Tunnel)

A live temporary tunnel is running at:

`https://witty-doodles-exist.loca.lt`

This tunnel only stays active while local processes are running.
